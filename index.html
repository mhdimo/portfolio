  • Badge Weighting System V2

    BWS-V2 or Badge Weighting System V2 is a system that i've designed in the past weeks for the game osu!.
    The formula consist of putting an age (expressed in integer) to badges awarded in the time span of 1 year subdiving them with weights in the following way:

    • from 0 to 3 months, the weight applied is 0.9;
    • from 3 to 6 months, the weight applied is 0.6;
    • from 9 to any months, the weight applied is 0.2;

    NOTE: this project is archived on GH, I won't develop a better version.

  • Arius80-HE

    Arius80-HE is my first project regarding PCB design on Hall Effect keyboards.
    I wanted to get a new keyboard featuring the following features from Wooting:
